**MaxTenis** é uma aplicação web de e-commerce desenvolvida com HTML5, CSS3 e JavaScript vanilla, criada para oferecer uma experiência de compra premium e moderna. O projeto combina design sofisticado com interatividade completa, sem dependências externas pesadas.

A plataforma é ideal para lojas de tênis que desejam oferecer uma experiência digital imersiva, com animações fluidas, sistema de carrinho funcional e interface responsiva para todos os dispositivos.
-

### Catálogo de Produtos
-  Grid de produtos responsivo
-  Filtro por marca (Nike, Adidas, Puma, etc)
-  Visualização rápida com modal
-  Avaliações de produtos
-  Preço original e promoção
-  Badges de "Novo" e "Destaque"
-  Seleção de tamanho

### Carrinho de Compras
-  Adicionar/remover produtos
-  Contador de itens
-  Cálculo automático de total
-  Sidebar animada
-  Toast notifications
-  Persistência de dados (localStorage)

### Busca e Filtro
-  Busca por nome de produto
-  Busca por marca
-  Filtros ativos destacados
-  Resultados em tempo real
-  Animação de fade-in dos resultados

### Favoritos (Wishlist)
-  Toggle de favoritos
-  Visual feedback imediato
-  Persistência de dados

### Oferta Relâmpago
-  Countdown timer animado
-  Dias, horas, minutos e segundos
-  Atualização em tempo real

### Newsletter
-  Formulário de inscrição
-  Validação de email
-  Toast de confirmação

###  Efeitos Especiais
-  Fundo com canvas de partículas animadas
-  Gradientes fluidos
-  Sombras dinâmicas
-  Transições suaves
-  Efeitos de hover sofisticados

**MaxTenis** é um projeto de e-commerce moderno, funcional e pronto para produção. Com JavaScript vanilla, design sofisticado e experiência do usuário excepcional, serve como excelente portfólio e base para projetos maiores.

Divirta-se explorando, personalizando e expandindo! 🚀
